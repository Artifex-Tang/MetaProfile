## 范围

### 标识

本测试说明适用于 MetaProfile 产业技术情报系统 V1.0。系统标识：名称 MetaProfile 产业技术情报系统，标识号 NRG00X/MetaProfile，版本号 V1.0。本文档对应的被测软件版本与上述标识一致。

### 系统概述

MetaProfile 是面向技术、项目、机构、人员四类实体的产业技术情报系统，由前端（React + nginx，端口 80）与单一复合后端（FastAPI，端口 8000）组成，经 Docker Compose 编排，默认启动全部服务。系统提供画像采集、检索、关联分析，以及扫描监测、新技术发现、选题服务、系统配置等功能。前端九个功能页面（总览、技术/项目/机构/人员画像、扫描监测、新技术发现、选题服务、系统配置）的可达交互、复合后端接口、前端单元逻辑构成本次测试对象。

### 测试环境构建

测试环境由 Docker Compose 一键拉起，拓扑与生产一致。构建步骤：

```bash
cd MetaProfile/deploy
cp .env.example .env
docker compose up -d --build
```

常驻容器须达 `(healthy)`：postgres、elasticsearch、redis、neo4j、rabbitmq、litellm、backend、frontend 共 8 个；`migrate`、`seed` 为一次性任务，`exited (0)` 属正常。前端单元测试环境：

```bash
cd frontend
npm ci
```
