### 系统概述

MetaProfile 由前端（React + nginx，端口 80）与单一复合后端（FastAPI，端口 8000）组成，经 Docker Compose 编排，提供技术/项目/机构/人员四类画像的采集、检索、关联分析，以及扫描监测、新技术发现、选题服务、系统配置等功能。本次测试依据《MetaProfile 系统测试说明》与《MetaProfile 软件用户手册》执行，覆盖前端九个功能页面的可达交互、关系图谱跨画像跳转新功能与前端单元逻辑。
